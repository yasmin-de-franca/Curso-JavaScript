Condições pt1

Sequência

Não existe condições/birfucações, tem um fluxo certo a ser seguido. Não pode-se se executar apenas o primeiro e o último comando.

Condições

Se for esse caminho, esxecuta-se esse comando, senão for esse caminho, executa-se outro comando. Existem duas possibilidades. Primeiro desvio - desvio condicional.
if (condição), {} = bloco, else = senão + {} bloco

if ( condição ){
true
} else {
false
}

Tipos de condição

Condição Simples

Apenas um bloco que será o verdadeiro
if ( condição ){
true
}

Condição Composta

Existem dois blocos, true e false
if ( condição ){
true
} else {
false
}

Condição Aninhada

Pega-se uma condição composta e coloca-se outras condições dentro dela.
if (cond1) {
bloco1
} else {
if (cond2) {
bloco 2 } else {
bloco3 }
}

Condição Múltipla

Possibilidade de vários. Uma expressão e testa-se valores fixos, e assim segue o fluxo comum.
switch (expressão) {
case valor 1:
break
case valor 2:
break
case valor 3:
break
default:
break
}
