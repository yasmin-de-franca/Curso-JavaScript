Condições

Se for esse caminho, esxecuta-se esse comando, senão for esse caminho, executa-se outro comando. Existem duas possibilidades. Primeiro desvio - desvio condicional.
if (condição), {} = bloco, else = senão + {} bloco

if ( condição ){
true
} else {
false
}
